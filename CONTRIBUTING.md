# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada Gogofit!

## Cara Kontribusi

1. Fork repositori
2. Buat branch baru
3. Commit dan push perubahan
4. Buat Pull Request

## Standar Coding

- Ikuti gaya masing-masing stack (PSR-12, PEP8, Dart Style)
- Tambahkan komentar jika perlu
- Jangan push ke `main` langsung
